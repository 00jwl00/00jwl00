liunx系统的常用指令

发表于 2017-08-09

常用命令

命令格式：命令 选项 参数

例：ls 【选项】 【文件或目录】
选项：-a 显示所有文件
-l 显示详细信息
-d 查看目录属性
-h 人性化显示文件大小
-i 显示inode

文件处理：

1、目录处理命令

建立：建立目录 mkdir [目录名]
递归创建 mkdir -p 先目录/后目录
切换：cd /目录/
cd -返回上一级
cd ..进入上一级目录
cd .进入当前目录
cd /按两下tab进入根目录下所有目录
相对路径：参照当前所在位置，当当前位置发生变
化时就不一定可以执行当前命令

绝对路径：从根目录下执行可以一直进行
pwd:打印当前目录
删除目录：只能删除空白目录 rmdir
rm [目录]然后根据提示输入
rm -rf 目录 强制删除
rm -r 目录 分级删除
复制：cp [选项][原文件或目录][目标目录]
cp -r 复制目录
cp -p 连带文件属性复制
cp -d 若源文件是连接文件，则复制链接属性
cp -a 相当于 -pdr所有属性都一样包括日志时间
剪切：mv [原文件或目录] [目标目录]
当原文件和目标文件在同一个目录下
用 mv 原文件目录 目标目录可改名
当原文件和目标文件不在同一个目录下即为剪切

2、文件处理命令

3、连接命令

软连接：ln -s[源文件][目标文件]
软连接更改源文件和目标文件
都会使文件改变，像windows里边的快捷方式
删除源文件会使连接断裂无法打开
硬连接：具有相同的i节点和存储空间

4、文件搜索命令

文件搜索命令 locate（定位）文件名 速度快
更新数据库：updatedb
权限问题需要加sudo
速度快 功能单一 只能按文件名搜索

命令搜索命令 whereis（哪里）与which（哪一个）
文件搜索命令 find（找）遍历 强大
格式 find [搜索范围][搜索条件] 具体查询
例 find / -name install.log
搜索范围缩小
(模糊查询)通配符 1） 匹配任意内容
2）[]匹配任意一个中括号内的内容
例： find /root -name “ab[cd]”
代表在root文件下搜索ab后边跟着一个c或者d的文件
如果仅有”*[cd]”代表搜索以c或者d结尾的文件
-user按照所有者搜索
-nouser查找没有所有者文件
-mtime 后边+或-或者只有一个数字代表查询那个数字
的前后或者当天的文件
-size 按照文件大小
find /etc -size +20k -a -size -50k代表查询文件大小在20-50k之间的文件
find /etc -size +20k -a -size -50k -exec ls -lh {}\;

---------------------------------- ------------  ---
          命令一                      命令二       标准格式
上式代表命令一的结果让命令二再执行

 -inum 直接按照文件接点查询(和mtime相似)

 3）？匹配一个单独的内容

字符串搜索 grep
格式 grep [选项] 字符串 文件名
选项：-i 忽略大小写
-v 排除指定字符串
例：grep “size” 文件名 代表搜索文件中具有size的文件
选项加在字符串前面

区别find与grep：find完全查询 grep包含查询
find搜索符合条件的文件
grep搜索符合条件的字符串

5、帮助命令

man 命令
help 命令（获取内部命令）
info 命令（会进入所有的子帮助界面，带有*号的为帮助条）
-u 进入上一层页面
-n 进入下一个帮助小节
-p 进入上一个帮助小节
-q 退出

6、压缩与解缩命令

.zip：
常用的压缩格式 .zip .gz .bz2
压缩文件.zip：zip 压缩文件名 源文件
压缩目录.zip: zip -r 压缩文件名 源目录
解压文件 unzip 压缩文件
.gz
压缩格式 gzip 文件名（原文件会消失）
gzip -c 源文件 > 压缩文件（原文件不消失）
gzip -r 目录（压缩目录下的子文件）
接缩文件 gzip -d 压缩包名
gunzip 压缩文件
gunzip -r 文件名 （解压目录下的子文件）
.bzip2
压缩格式 bzip2 源文件(原文件消失)
bzip2 -k 源文件（保留原文件）
不能压缩目录
解缩文件 bzip2 -d 压缩文件
bunzip2 压缩文件
加-k会保留原压缩文件
压缩目录的方法：
.tar.gz
压缩格式 打包：tar -cvf 打包文件名 源文件
压缩：任何一种想用的压缩格式 打包文件名
解缩文件 解包：tar -xvf 打包文件名
直接压缩格式:
.tar.gz
.tar.gz : tar -zcvf 压缩包名.tar.gz 源文件
解缩文件
tar -zxvf 压缩包名.tar.gz
.tar.bz2
tar -jcvf 压缩包名.tar.bz2 源文件
解缩文件
tar -jxvf 压缩包名.tar.bz2
手动解压缩位置 ：tar -jxvf jp.tar.bz2 -C 目录名
一次压缩多个目录：tar -zcvf 压缩包名.tar.gz 目录名1 目录名2
若想特定压缩位置：在压缩包名前面加绝对路径即可（目标目录名）
只查看不解压 tar -ztvf 压缩包名.tar.gz

7、关机与重启

shutdown [选项] 时间
选项分为-c:取消前一个关机命令
-h:关机
-r:重启 时间后边加&，会在后台执行
关机 halt poweroff init 0（不用）
重启 reboot init 6（不用）
退出登录logout

8、shell脚本执行方式

输出指定内容：echo 选项 输出内容
-e：反斜杠控制字符（在要操作的地方直接放置控制字符）
\a输出警告音\b退格删除键\n换行符\r回车键
\t制表符tab键\v垂直制表符\0nnn按照8进制输出
\xhh十六进制输出（hh&nnn都是数字）
颜色输出”\e[1;颜色代码 输出内容 \e[0m”即可输出指定内容
30m=黑色，31m=红色，32m=绿色，33m=黄色
34m=蓝色，35m=洋红，36m=青色，37m=白色

*PATH:定义的是系统搜索命令的路径。

*在Ubuntu中下载软件包 sudo apt-get install vim
