Flask学习（1）

发表于 2018-07-29 | 阅读次数

第一次创建flask项目需要选择选择虚拟环境位置：virtualenv/flask-env/bin/python

from flask import Flask//从flask这个框架导入Flask这个类
app = Flask(name)//初始化flask对象：flask（）括号里边需要传递一个参数name(固定格式)
@app.route(‘/‘)//是一个装饰器：@开头，并且在函数上面，说明是一个装饰器
def hello_world()://装饰器的作用：是做一个url与视图函数的映射可以理解为：在127.0.0.1：5000/端口去请求这个函数然后将结果
return ‘Hello World!’//返回给浏览器
if name == ‘main‘://如果当前这个文件是作为入口程序来运行，那么就执行app.run()
app.run()//启动一个应用服务器，来接受用户的请求
//相当于while true:
//设置debug模式：在app.run()中传入一个参数debug：app.run(debug=True)即可
//该模式的两大功能和好处：1直接在页面就能看到出错的地方2实时更新
//使用配置文件设置debug模式：新建config.py文件，并且配置到主文件中，代码如下：
、、、
import config
app.config.from_object(config)
